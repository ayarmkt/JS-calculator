# JavaScript Calculator

## Overview

A calculator made with JavaScript. The calculator displays both the calculation process and the result. It has a plus/minus button and a percentage button.

## Tech Stack
Tech Stack: HTML | CSS | JavaScript<br>
Platform: GitHub Pages

## Final Product

**Live Site:** https://ayarmkt.github.io/JS-calculator/

**Calculation**<br>
A calculator that displays the calculation process and result. This calculator can calculate continuously even after pressing "=".
